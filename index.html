Inspetor de Elementos

Em qualquer site é possível verificar a estrutura do HTML. Para isso, basta usar a tecla F12 no site, ou usar o botão direito e selecionar a opção “Inspecionar”. No menu da estrutura do HTML tem a função “Select an element in the page to inspect it" (CTRL+SHIFT+C), que permite que selecionar uma área específica do site, e verificar sua estrutura.

Estrutura Básica do HTML

A estrutura básica do HTML é composta por 3 tags:

  • < HTML > < /HTML > - Tudo que estiver entre essa tag, será entendido pelo navegador como um HTML.
  • < Head > < /Head > - São as informações pré carregadas para o usuário, como o título que irá aparacer na aba da página do navegador.
  • < Body > < /Body > - É todo o contéudo que irá aparacer na página do navegador para o usuário, como textos, imagens, etc.

Falando Sobre Tags

As tags HTML são utilizados para informar ao navegador que tipo de estrutura está sendo construída, podendo ser títulos, parágrafos, imagens, links, etc. Normalmente, as tags são compostas por abertura < > e fechamento < / > . Abaixo alguns exemplos de tags:

  • < i > Itálico < / i >
  • < strong > Negrito < / strong >
  • < input type: "text" > Cria um campo de digtitação de texto e < input type: "number" > cria um campo de digitação númerico: (obs: essa tag não tem fechamento).

Atributos das Tags

Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em todas as Tags, como: Id, Style e Class) e específicos (que são direcionados para cada Tag, através de especificação, como: Type para a tag input, e src e width para a tag img).

Textos

As principais tags para textos são:

  • < h1 >, < h2 >, < h3 >, < h4 >, < h5 > e < h6 >: Determinam o tamanho dos títulos e subtítulos, sendo h1 o maior e h6 o menor.
  • < p > Parágrafo < / p >
  • < blockquote >
    Citação, observação
    < / blockquote >
  • < u >Sublinhado < / u >
  • < mark > Marca texto < / mark >
  • < sup > Sobrescrito < / sup >
  • < sub > Subscrito < / sub >
  • < del > Texto excluído < / del >

Listas Ordenadas e Não Ordenadas

  • As Listas Ordenadas seguem uma sequencia númerica:

< ol > < / ol > junto com a tag "filha" < li > < / li >

Ex:

  1. Linha A
  2. Linha B
  3. Linha C

  • As Listas Não Ordenadas utilizam marcadores:

< ul > < / ul > junto com a tag "filha"< li > < / li >

Ex:

  • Linha A
  • Linha B
  • Linha C

obs: Tanto nas Listas Ordenadas, como nas Não Ordenadas, todas as linhas precisam usar a tag < li > < /li >

Links

A tag para adicionar links é escrita da seguinte forma: < a href = "Link do site" > "nome do link" < / a > . Essa tag possuí atributos como: Targer e title. Além disso, é possível colocar o nome de outras pagínas do próprio site como link.
